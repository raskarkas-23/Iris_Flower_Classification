# 🌸 Iris Flower Classification System

This project is a Machine Learning web application that predicts the species of an Iris flower using a Random Forest model.

---

## 📌 Project Description

The user enters:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The system predicts the flower species and displays its image.

---

## 🧠 Machine Learning Algorithm

Random Forest Classifier (Scikit-learn)

---

## 💻 Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML & CSS
